Web开发常用Javascript方法集（webutils）
==========================

##Url参数获取方法

###获取url参数

* 方法：

  webutils.url.get(name,url)

* 参数：

  name：url参数名。

  url：(可选)，默认是当前url。

###设置url参数

* 方法：

  webutils.url.set(name,value,url)

* 参数：

  name：url参数名。如果name不存在则新增，反之则更新。

  value：url参数值。

  url：(可选)，默认是当前url。
